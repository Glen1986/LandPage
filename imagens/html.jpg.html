Cómo redirigir o redireccionar a otra página web en HTML

Programación HTML  
 

Al diseñar y desarrollar nuestra página web, siempre, en algún momento, nos surge la necesidad de hacer que el usuario sea redirigido a otra página web, bien porque la actual ha dejado de ser operativa, o bien porque momentáneamente deseamos que este usuario vea cualquier tipo de información.

Normalmente se usa lenguaje del lado del servidor, como PHP o ASP y, en otras ocasiones, también suelen usarse Javascript para hacer este redireccionamiento. El problema radica en que si el visitante está navegando por nuestro sitio web mediante un navegador que tiene deshabilitado los scripts, esta última solución no será factible, por lo que una opción es hacer que el redireccionamiento se haga desde el propio HTML.

 

Cómo redireccionar a otra página web mediante HTML

 

Existe un método para informar a un navegador web de que automáticamente refresque el contenido de la página actual o de un frame en un intervalo de tiempo. Para ello, se usa el atributo http-equiv de la etiqueta META. Este atributo se configura con el valor ‘refresh‘ y en el atributo content es donde se sitúa el intervalo de tiempo, en segundos, en el que queremos que se realice ese refresco.

Si además, añadimos a este atributo el valor url, haremos que, en el tiempo indicado, la página web redirija al usuario hacia la dirección URL que indiquemos en el mismo. Vamos a verlo con un ejemplo:

XHTML
1
<meta http-equiv="refresh" content="0; url=http://www.nuevaURL.com"/>

Como puedes ver, con esta línea de código estamos indicando al navegador que directamente redirija la página actual a la nueva URL indicada en el valor url del atributo content, evitando el uso de cualquier tipo de lenguaje de script como Javascript.

Así, podemos estar seguros de que el usuario será redirigido a la nueva URL aún teniendo el navegador de este deshabilitada la ejecución de scripts, aunque también hay que tener en cuenta que no todos los navegadores soportan esto.
