Cmo redirigir o redireccionar a otra pgina web en HTML

Programacin HTML  
 

Al disear y desarrollar nuestra pgina web, siempre, en algn momento, nos surge la necesidad de hacer que el usuario sea redirigido a otra pgina web, bien porque la actual ha dejado de ser operativa, o bien porque momentneamente deseamos que este usuario vea cualquier tipo de informacin.

Normalmente se usa lenguaje del lado del servidor, como PHP o ASP y, en otras ocasiones, tambin suelen usarse Javascript para hacer este redireccionamiento. El problema radica en que si el visitante est navegando por nuestro sitio web mediante un navegador que tiene deshabilitado los scripts, esta ltima solucin no ser factible, por lo que una opcin es hacer que el redireccionamiento se haga desde el propio HTML.

 

Cmo redireccionar a otra pgina web mediante HTML

 

Existe un mtodo para informar a un navegador web de que automticamente refresque el contenido de la pgina actual o de un frame en un intervalo de tiempo. Para ello, se usa el atributo http-equiv de la etiqueta META. Este atributo se configura con el valor ‘refresh‘ y en el atributo content es donde se sita el intervalo de tiempo, en segundos, en el que queremos que se realice ese refresco.

Si adems, aadimos a este atributo el valor url, haremos que, en el tiempo indicado, la pgina web redirija al usuario hacia la direccin URL que indiquemos en el mismo. Vamos a verlo con un ejemplo:

XHTML
1
<meta http-equiv="refresh" content="0; url=http://www.nuevaURL.com"/>

Como puedes ver, con esta lnea de cdigo estamos indicando al navegador que directamente redirija la pgina actual a la nueva URL indicada en el valor url del atributo content, evitando el uso de cualquier tipo de lenguaje de script como Javascript.

As, podemos estar seguros de que el usuario ser redirigido a la nueva URL an teniendo el navegador de este deshabilitada la ejecucin de scripts, aunque tambin hay que tener en cuenta que no todos los navegadores soportan esto.
